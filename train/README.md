## Weight and Bias Setup

Set the `WANDB_API_KEY` enviroment variable using `api key` the from your account in `wandb`.
 
```bash
# from bash
export WANDB_API_KEY=<your api key>

# or using conda
conda env config vars set WANDB_API_KEY=<your api key>
```
